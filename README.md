# PracticePython-2P4

Программа находит шаблон формы исходя из полученных данных и валидирует их по типам

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py get_tpl --customer=John Smith --дата_заказа=27.05.2025
```

## Тесты

```bash
pytest test_app.py
```
